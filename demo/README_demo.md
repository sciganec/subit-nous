# SUBIT-NOUS Demo Folder

This folder contains example files covering all 4 transversal modes:

- **micro/** ? MICRO mode (ME, EAST, SPRING, LOGOS)
- **macro/** ? MACRO mode (WE, SOUTH, SUMMER, ETHOS)
- **meso/** ? MESO mode (YOU, WEST, AUTUMN, PATHOS)
- **meta/** ? META mode (THEY, NORTH, WINTER, THYMOS)
- **mixed/** ? files with transitions between modes
- **code/** ? source code (emphasizes LOGOS)
- **images/** ? placeholders for image analysis

Run:

    nous analyze demo --output demo_output

Then open demo_output/graph.html to see the knowledge graph.
